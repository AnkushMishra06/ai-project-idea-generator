# AI Project Idea Generator

## Overview
Students/early-Developers struggle to find ideas for building projects for their portfolio or resumes, sometimes the brainstormed
ideas are very general projects, very large implementation timeline, not aligned with career or copied from tutorials. After getting
overwhelmed by these ideas they are left only tinkering on their choices or watching unlimited tutorials giving no personalized growth
This project is intended to help those developers by filling the gap between learning and skill based growth.
---

## Problem Solved
This project will solve the problem of developers scrolling through internet, blog lists, ChatGPT prompts and still
not finding good project ideas, by taking their skills and level of expertise to recommend them project ideas that will
get them hired.
---

## Tech Stack
- Backend - __FastAPI__  for logic, fast implementation and LLM orchestration
- Frontend - __Streamlit__ for easy frontend building.
- LLM API - *to be done in future*
- Database - __PostgreSQL__ for user profiles, saving ideas and feedback from user
- Deployment - __Render__
